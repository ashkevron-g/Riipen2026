# Riipen2026
Capstone project in Fanshawe Data Analytics for real world company Ada Analytics 

## About the Project: 
The class used Riipen to connect with real world companies to help solve buisness issues with our data analytics skills. My group was paired with Ada Analytics. 
Ada Analytics is a consulting firm wished to get a better idea of where their revenue was coming from, and how to optimize time spent on their various clients

## My Role: 
My role was data wrangler. I took the raw data that Ada Analytics was already using and prepared it for analytics. I standardized, cleaned, reformated, and recalculated the provided data in Excel.
I created the data dictionary so that Ada Analytics could continue to build from the model the group created.

## Key Skills & Tools: 
* Excel forumaltion
* Power BI Dax
* Busines valuation
* Data engineering
* Data pipelines

## How to Navigate the Repo: 
My main contribution for the capstone project was creating the 'Data Management.docx' document. Input from my team helped me decide what was key information, and how best to format things for our dashboard analytics. 
It lead to my creating the 'Consulting Data Dictionary.xlsx'. This, along with the cleaned up dataset, was the jumping off point for my team. 
'Riipen certificate.pdf' is the certification of my completion of this capstone project.
